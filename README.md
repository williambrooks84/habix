Ce projet [Next.js](https://nextjs.org) a été créé avec [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app). La stylisation a été réalisée avec [Tailwind CSS](https://tailwindcss.com/).

## Démarrage

Pour commencer, lancez le serveur de développement :

```bash
npm run dev
```

Ouvrez [http://localhost:3000](http://localhost:3000) dans votre navigateur pour voir le résultat.

## Déploiement sur Vercel

La branche principale (main) de ce projet est déployée en continu sur Vercel : [Voir le projet](https://habix-three.vercel.app)
